# NSURLProtocol

简介：它是一个抽象类，提供了加载URL数据需要执行的特定协议的基本结构。

注意：

* 应用程序不要直接实例化`NSURLProtocol`的子类，在`NSURLConnection`开始下载时会创建`NSURLProtocol`子类的实例。
